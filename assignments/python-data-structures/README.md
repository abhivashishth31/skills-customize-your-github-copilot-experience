# 📘 Assignment: Data Structures in Python

## 🎯 Objective

Master Python's core data structures—lists, dictionaries, sets, and tuples—and learn how to choose the right data structure for different programming tasks. You'll implement practical functions that demonstrate when and how to use each structure effectively.

## 📝 Tasks

### 🛠️ Working with Lists and Tuples

#### Description
Create functions that demonstrate list and tuple operations, including indexing, slicing, sorting, and when to use immutable tuples over mutable lists.

#### Requirements
Completed program should:

- Create a function that finds the maximum and minimum values in a list
- Create a function that removes duplicates from a list while preserving order
- Create a function that flattens a nested list into a single-level list
- Create a function that merges two sorted lists into one sorted list
- Create a function that demonstrates the immutability of tuples
- Include proper docstrings explaining when to use tuples vs lists
- Handle edge cases (empty lists, None values, mixed data types)

### 🛠️ Mastering Dictionaries

#### Description
Implement functions that work with dictionaries for efficient key-value pair storage and retrieval, including nested dictionaries and dictionary operations.

#### Requirements
Completed program should:

- Create a function that counts the frequency of items in a list and returns a dictionary
- Create a function that merges two dictionaries while handling key conflicts
- Create a function that inverts a dictionary (swap keys and values)
- Create a function that finds all keys with a specific value
- Create a function that works with nested dictionaries (e.g., accessing deeply nested values)
- Sort dictionaries by keys or values
- Include error handling for missing keys and type mismatches

### 🛠️ Sets and Data Structure Selection

#### Description
Explore set operations for efficient membership testing and build a comprehensive function that chooses the optimal data structure for different scenarios.

#### Requirements
Completed program should:

- Create a function that finds common elements between multiple lists using sets
- Create a function that finds the difference between two sets
- Create a function that removes duplicates using set operations
- Create a function that checks if all elements in one list are in another
- Demonstrate the performance advantages of sets for membership testing
- Create a function that recommends which data structure to use based on a given scenario
- Compare the time complexity of different data structure operations
