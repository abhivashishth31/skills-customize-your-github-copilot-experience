# 📘 Assignment: Python Text Processing

## 🎯 Objective

Master string manipulation, file I/O operations, and text processing techniques in Python. You'll build practical text processing utilities that read, analyze, and transform text files, gaining essential skills for working with real-world data.

## 📝 Tasks

### 🛠️ String Manipulation and Analysis

#### Description
Create functions that analyze and manipulate text strings, implementing common string operations and transformations.

#### Requirements
Completed program should:

- Create a function that counts the frequency of words in a text string (case-insensitive)
- Create a function that reverses the order of words in a sentence while preserving punctuation
- Create a function that removes duplicate characters while maintaining order
- Return results in appropriate data structures (dictionaries, lists, or strings)
- Handle edge cases (empty strings, special characters, whitespace)
- Include clear docstrings explaining each function's purpose

### 🛠️ File I/O Operations

#### Description
Implement functions that read, write, and process data stored in files using proper file handling techniques.

#### Requirements
Completed program should:

- Create a function that reads a text file and returns its contents line by line
- Create a function that writes processed text to a new file
- Implement proper file handling using `with` statements to ensure files are closed correctly
- Create a function that counts lines, words, and characters in a file
- Handle file not found errors and other I/O exceptions gracefully
- Include proper error messages when files cannot be accessed

### 🛠️ Text Processing and Transformation

#### Description
Build a comprehensive text processing tool that combines string manipulation and file operations to transform text data.

#### Requirements
Completed program should:

- Read text from an input file
- Apply text transformations (convert case, remove extra whitespace, filter specific content)
- Generate a summary report with statistics (word count, unique words, character frequency)
- Write processed text and report to output files
- Support multiple text processing operations chained together
- Display clear progress or results to the user
