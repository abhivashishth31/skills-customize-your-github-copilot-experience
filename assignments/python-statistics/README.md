# 📘 Assignment: Statistics with Python

## 🎯 Objective

Master statistical analysis and data manipulation using pandas and numpy libraries. You'll learn to perform descriptive statistics, hypothesis testing, data aggregation, and exploratory data analysis to extract meaningful insights from datasets.

## 📝 Tasks

### 🛠️ NumPy Fundamentals and Array Operations

#### Description
Build foundational skills with NumPy arrays, understanding how to create, manipulate, and perform mathematical operations efficiently on numerical data.

#### Requirements
Completed program should:

- Create and manipulate NumPy arrays (1D, 2D, and multidimensional)
- Perform basic statistics (mean, median, standard deviation, variance)
- Use NumPy functions for mathematical operations (sum, product, dot product)
- Apply array indexing, slicing, and reshaping techniques
- Generate random numbers and random samples from distributions
- Perform element-wise operations and broadcasting
- Calculate percentiles and quartiles
- Handle edge cases (empty arrays, NaN values, inf values)

### 🛠️ Pandas DataFrames and Data Manipulation

#### Description
Learn to work with pandas DataFrames for efficient data loading, cleaning, filtering, and transformation tasks commonly used in data analysis.

#### Requirements
Completed program should:

- Load data from CSV, JSON, and other file formats
- Create DataFrames and Series objects from various data sources
- Filter and subset data using conditions
- Handle missing values (detection, removal, and imputation)
- Group data by categories and perform aggregations
- Sort and rank data
- Merge and join multiple DataFrames
- Create pivot tables and cross-tabulations
- Apply custom functions to DataFrame columns

### 🛠️ Statistical Analysis and Hypothesis Testing

#### Description
Perform comprehensive statistical analyses including descriptive statistics, distributions, correlations, and hypothesis testing to draw conclusions from data.

#### Requirements
Completed program should:

- Calculate descriptive statistics (mean, median, mode, range, IQR)
- Analyze distributions (skewness, kurtosis, normality tests)
- Calculate correlations and covariance between variables
- Perform basic hypothesis tests (t-tests, chi-square tests)
- Create statistical summaries and reports
- Visualize statistical distributions and correlations
- Identify and handle outliers
- Generate confidence intervals
