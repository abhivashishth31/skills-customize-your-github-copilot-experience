# 📘 Assignment: Python Data Visualization

## 🎯 Objective

Learn to create professional, informative data visualizations using matplotlib and plotly libraries. You'll transform raw data into compelling charts, graphs, and interactive dashboards that effectively communicate insights and patterns.

## 📝 Tasks

### 🛠️ Creating Basic Visualizations with Matplotlib

#### Description
Build foundational plotting skills by creating various chart types to visualize different kinds of data.

#### Requirements
Completed program should:

- Create line plots to show trends over time
- Create bar charts to compare categorical data across groups
- Create scatter plots to visualize relationships between variables
- Create histograms to show data distributions
- Add proper labels, titles, and legends to all charts
- Customize colors, line styles, and marker styles
- Save visualizations to image files (PNG, PDF)
- Use subplots to display multiple plots in a single figure

### 🛠️ Advanced Matplotlib Techniques and Customization

#### Description
Enhance your visualization skills with advanced styling, multiple plot types, and specialized chart configurations.

#### Requirements
Completed program should:

- Create pie charts with proper formatting and exploded slices
- Create box plots to show statistical distributions and outliers
- Create violin plots for detailed distribution comparison
- Create heatmaps to visualize multidimensional data
- Apply custom themes and color palettes
- Add annotations and text elements to charts
- Create subplots with different chart types
- Handle edge cases (empty data, missing values, outliers)

### 🛠️ Interactive Visualizations with Plotly

#### Description
Build interactive, web-ready visualizations using plotly that allow users to explore data dynamically.

#### Requirements
Completed program should:

- Create interactive line charts with hover tooltips
- Create interactive bar charts with click and zoom capabilities
- Create interactive 3D scatter plots
- Create animated charts showing how data changes over time
- Build interactive dashboards combining multiple charts
- Export interactive visualizations as HTML files
- Use Plotly Express for quick prototyping
- Add filtering and selection options for data exploration
